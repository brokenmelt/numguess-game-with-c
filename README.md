# numguess-game-with-c
#Number Guessing Game in C

Description

This is a simple Number Guessing Game written in C. The computer generates a random number between 1 and 100, and the player keeps guessing until the correct number is found.

Features

- Random number generation
- User input using "scanf()"
- "if", "else if", and "else" conditions
- "while" loop
- Displays whether the guess is too high or too low
- Congratulates the player when the correct number is guessed

Concepts Used

- Variables
- "printf()" and "scanf()"
- "if-else"
- "while" loop
- "rand()"
- "srand()"
- "time()"

Libraries Used

- "stdio.h"
- "stdlib.h"
- "time.h"

How to Run

1. Compile the program using a C compiler.
2. Run the executable.
3. Enter your guesses until you find the correct number.

Future Improvements

- Count the number of attempts
- Limit the number of guesses
- Add a Play Again option
- Add difficulty levels (Easy, Medium, Hard)
